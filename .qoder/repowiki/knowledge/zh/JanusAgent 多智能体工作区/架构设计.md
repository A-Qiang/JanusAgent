根目录作为 uv workspace 入口：pyproject.toml 通过 [tool.uv.workspace] members = packages/* 声明所有子包，并在 [dependency-groups.dev] 与 [tool.uv.sources] 中以 workspace=true 将子包作为本地可编辑依赖引入；main.py 作为顶层可执行脚本，import quant_agent 与 companion_agent 并调用各自的 hello() 完成双智能体串联。packages/ 下每个子包独立维护 pyproject.toml 与源码，彼此通过 Python import 而非进程间通信耦合，docs/ 存放面向人类阅读的非代码资产。
