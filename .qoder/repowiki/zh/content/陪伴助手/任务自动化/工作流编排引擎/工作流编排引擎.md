## 简介
本技术文档面向“陪伴助手”的工作流编排引擎，聚焦以下关键能力：
- 任务依赖解析算法：有向无环图（DAG）构建、拓扑排序与循环依赖检测
- 条件分支处理机制：决策树实现、条件表达式解析与动态路径选择
- 异常恢复机制：错误分类、重试策略、补偿事务与回滚操作
- 状态持久化与断点续跑：工作流状态存储、检查点与恢复流程
- 版本管理与向后兼容：工作流版本控制与兼容性保证策略

本项目仓库包含工作流校验脚本与规范说明，以及应用入口与基础数据模型。基于这些材料，本文对编排引擎的算法与工程实践进行系统化梳理，并提供可落地的架构图与流程图。

## 详细组件分析

### 任务依赖解析算法（DAG 构建、拓扑排序、循环检测）
- DAG 构建
  - 将节点作为顶点，边作为依赖方向，形成有向图
  - 校验层确保源/目标节点存在且句柄格式正确
- 拓扑排序
  - 使用入度表与队列进行 Kahn 算法
  - 若队列空且仍有未访问节点，则存在环
- 循环依赖检测
  - 在 DFS 过程中记录回溯边，仅在含 loop 节点的子图中允许环
  - 否则判定为非法循环并报错

```mermaid
flowchart TD
Start(["开始"]) --> Build["构建邻接表与入度表"]
Build --> InitQ["将所有入度为0的节点加入队列"]
InitQ --> Loop{"队列是否为空?"}
Loop --> |否| Pop["弹出节点u<br/>加入拓扑序列"]
Pop --> ForEach["遍历u的邻居v"]
ForEach --> Dec["v的入度减1"]
Dec --> Zero{"v入度是否为0?"}
Zero --> |是| Enq["将v加入队列"]
Zero --> |否| NextV["继续下一个邻居"]
Enq --> NextV
NextV --> Loop
Loop --> |是| CheckAll{"拓扑序列长度等于节点数?"}
CheckAll --> |是| Done["完成(无环)"]
CheckAll --> |否| Cycle["检测到环(非法)"]
```

### 条件分支处理机制（决策树、表达式解析、动态路径）
- 决策树实现
  - 每个条件节点维护若干分支，分支条件为布尔表达式
  - 运行时根据上下文变量求值，选择唯一分支进入下一节点
- 条件表达式解析
  - 支持简单比较与逻辑组合，必要时引入安全沙箱环境
  - 变量来源包括上游节点输出、系统配置与用户输入
- 动态路径选择
  - 根据分支结果动态决定后续执行路径
  - 结合拓扑排序，确保所选路径仍满足无环约束

```mermaid
flowchart TD
Enter(["进入条件节点"]) --> Eval["解析并求值分支条件"]
Eval --> Branch{"是否命中某分支?"}
Branch --> |是| Select["选择对应下游节点集合"]
Branch --> |否| Fallback["执行默认分支或返回错误"]
Select --> Merge["汇聚到下一阶段(保持DAG性质)"]
Fallback --> End(["结束/上报错误"])
Merge --> End
```

### 异常恢复机制（错误分类、重试、补偿与回滚）
- 错误分类
  - 可重试错误：网络抖动、临时资源不可用
  - 不可重试错误：参数非法、权限不足、业务校验失败
- 重试策略
  - 指数退避 + 抖动，限制最大重试次数
  - 针对幂等操作可安全重试；非幂等需加锁或去重
- 补偿事务与回滚
  - 对副作用操作记录补偿动作（如删除已创建资源）
  - 失败时按逆序执行补偿，保证最终一致性

### 状态持久化与断点续跑
- 状态模型
  - 全局状态：运行ID、版本、开始/结束时间、当前阶段
  - 节点状态：状态机（待执行/运行中/成功/失败/跳过）、输入输出快照、错误信息
  - 检查点：增量写入，包含最近一次稳定状态
- 断点续跑
  - 启动时加载最新检查点，恢复未完成的节点集
  - 对幂等节点可直接重放；非幂等节点通过幂等键避免重复副作用
- 一致性保障
  - 原子写入检查点（先写临时文件再替换）
  - 失败后保留旧检查点，避免损坏

## 性能考量
- 拓扑排序复杂度 O(V+E)，适合大规模工作流
- 并行执行需评估共享资源竞争与内存占用
- 检查点写入应批量合并，降低 I/O 压力
- 条件表达式求值尽量缓存中间结果，避免重复计算

## 故障排查指南
- 常见校验错误
  - 缺少必要节点：workflowStart 或输出节点缺失
  - 连接无效：source/target 不存在或句柄格式不正确
  - 连通性问题：存在不可达节点
  - 循环依赖：非 loop 节点构成的环
- 定位步骤
  - 查看校验报告中的错误级别与字段路径
  - 使用连通性检查确认从起点可达性
  - 对循环依赖，识别环上节点并重构流程
